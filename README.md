## Description
Dernier Souvenir est une histoire interactive centrée sur le deuil, la mémoire et la perception altérée. Le joueur accompagne David dans l’ascension d’une montagne, où chaque choix révèle une facette du traumatisme qu’il fuit.  

### Fonctionnalités principales
- Système complet de navigation de chapitres  
- Choix conditionnels et fins multiples (6 fins)  
- Système de “mémoire” permettant de stocker les choix influents  
- Interface narrative immersive avec contenu illustré  
- Gestion de l’historique des décisions  
- Responsive complet (mobile, tablette, desktop)  
- Accessibilité : navigation clavier, textes alternatifs, contraste conforme

## Comment jouer
1. Ouvrir la démo en ligne.  
2. Cliquer sur **Jouer** ensuite sur **Nouvelle Partie**.  
3. Lire la narration et choisir entre les différentes options.  
4. Des fins se débloquent lorsque vous terminez l'histoire (boucle).
5. Explorer les différentes branches.
6. Lorsque vous terminez le jeu, votre sauvegarde s'effacera.

## Installation en local pour développement
